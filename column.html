2023.01.06

移り変わる季節と、変わらない日常

季節の記憶 / 保坂和志

保坂和志さんの書く文章が大好きです。この作品も夢中になって読んでしまいました。

鎌倉の稲村ヶ崎が舞台で、シングルファーザーの中野とその息子のクイちゃんの日常が描かれた話。ページをめくってもめくっても、なにかしらのドラマが起こったり登場人物の関係が変わったりするわけではないのだけれど、この本に流れる雰囲気が終始好きだった。中野が見たまま感じたままにつらつらと書き綴る感じ。

日常として変わらぬ時間が過ぎていく中で季節がゆっくりと移り変わっていく様が、なんだか少し寂しく感じる作品です。

2022.1.04

人々が惹かれるデザインについて

GOOD DESIGN FILE / 高橋克典

長く親しまれる商品やブランドの魅力を解説し、デザインや経営のヒントになりそうな一冊。著者は、目まぐるしく流行の変わるブランドビジネスにおいて数々の企業を売上アップに導いてきた高橋克典さん。

家具や家電、普段身につけるものなどのあらゆるジャンルで、多くの人に愛され続ける製品を解説しまとめられた充実の内容でした。

優れたモノの背景には人の思い・企業の理念や目的が明確にあり、それらを目に見える形に変えること。人々が惹かれるモノにはそういったデザインがなされていることが理解できました。

新しい民藝のカタチ

わかりやすい民藝 / 高木崇男

この本を読んで「民藝＝用の美＝機能美」だけではない新しい側面を知ることができました。「民藝」とはもともと地域の人々の間にあった美を、政府から押し付けられる美の価値観から守るために生まれ、また決して確率した過去の概念ではなく今も存在し続けていることなど、民藝という概念が培ってきた過程についても勉強することができました。

2022.12.29

飲んだことのないウイスキーの味

もしも僕らのことばがウイスキーであったなら / 村上春樹

偶然聞いたTOKYO FMで、福山雅治さんの番組で紹介されていた本。

飲んだことのない異国のウイスキーの味と、行ったこともない土地の空気を想像できて読んでいて幸せな気持ちになりました。このエッセイを読み終えてから、留学生のころにいろんなヨーロッパの国を旅したことを思い出し、もう一度行きたいなと思った。あの頃よりはだいぶ成長しているだろうし、お酒の味もわかるような気がする。

こんな世の中になってしまったからこそ、すごくすごく恵まれた環境にいたんだなぁなどと、その土地を離れてから思うことがたくさんあることに気づけました。

優しさと、思考の柔軟さ

幸せについて / 谷川俊太郎

選ぶ言葉、文脈からその人の優しさと、思考の柔軟さが伝わる作品。幸せについて考えれば考えるほど難しいと思っていたけれど、「幸せだと気づかないくらいの、BGMみたいな幸せが一番確実な幸せ」という谷川俊太郎さんの言葉に、心の中のなにかがほどけるような感覚がありました。

自分がブレてしまいそうな時に

今日もていねいに 。 / 松浦弥太郎

自分がブレてしまいそうな時に読みたい本。

浅はかな考えで、焦ってなにかに取り組むのは良くない。ただ、ゆっくりやることが、丁寧なこととも限らない。何が正しいのかじっくり考えらる余裕がない、そんな時に読みたい本でした。
